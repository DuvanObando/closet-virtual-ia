# Closet Virtual IA — Sistema Experto Multimodal de Recomendación de Outfits

Sistema experto que analiza prendas de ropa mediante imágenes y recomienda outfits personalizados combinando un motor de reglas clásico con el modelo multimodal Gemini 1.5 Flash.

## Estado del proyecto

En desarrollo activo. Documentación completa próximamente.

## Stack tecnológico

- Python 3.10+
- Streamlit (dashboard)
- Google Gemini 1.5 Flash (análisis multimodal de imágenes)
- Sistema experto basado en reglas (forward chaining)
- Soporte multi-proveedor: Gemini, Groq, Ollama, HuggingFace
